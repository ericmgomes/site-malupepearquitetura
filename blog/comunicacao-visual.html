Comunicação visual

Tipo: post
Data: 2017-05-31T19:29:01
Atualizado: 2024-10-11T20:16:54
Link original: https://malupepearquitetura.com.br/comunicacao-visual/

Quando a comunicação visual é aplicada ao projeto de arquitetura e interiores, os ambientes ganham mais identidade. Com as cores, linhas e design os espaços comunicam-se melhor com os usuários.

É necessário que o usuário entenda o espaço que utiliza e assimile as informações nele contidas por meio de elementos gráficos.

O projeto de comunicação visual deve ser feito em paralelo ao projeto de arquitetura, em alguns casos o projeto de arquitetura não pode competir com a identidade visual de uma marca, ou vice-versa.

A comunicação visual deve estar totalmente em harmonia com o projeto de arquitetura, para que consiga comunicar o que se propõe.
